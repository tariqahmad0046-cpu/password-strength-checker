# 🔐 Password Strength Checker

## 📌 Description
A web-based password strength checker built using Flask. It evaluates password security and provides feedback to improve strength.

## 🚀 Features
- Detects weak, medium, strong passwords
- Suggests improvements
- Clean UI design

## 🛠️ Technologies Used
- Python
- Flask
- HTML/CSS

## ▶️ How to Run

1. Clone repository:
git clone https://github.com/your-username/password-strength-checker.git

2. Install dependencies:
pip install -r requirements.txt

3. Run:
python app.py

4. Open browser:
http://127.0.0.1:5000/
